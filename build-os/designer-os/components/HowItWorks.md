# HowItWorks

## Purpose
Guide visitors through the water treatment process with clear, numbered steps: Schedule → Test → Install → Enjoy.

## Content Zones
- Section header with title and subtitle
- 4 step cards/items:
  1. Schedule - Book your free consultation
  2. Test - We analyze your water quality
  3. Install - Professional installation
  4. Enjoy - Clean, healthy water

## Visual Style
- Horizontal timeline on desktop with connecting lines
- Vertical stack on mobile
- Numbered circular icons (1-4)
- Light zinc background with subtle card styling
- Icons or numbers with color accent

## Dependencies
- `@/components/ui/container` for consistent width
- lucide-react icons (Calendar, Droplet, Wrench, Smile)

## Component API
```tsx
<HowItWorks />
```

## Accessibility
- Semantic list structure
- Proper heading hierarchy (h2 for section, h3 for steps)
- Numbers/icons are decorative, text conveys meaning
