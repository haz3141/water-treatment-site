# CtaFooter

## Purpose
Encourage final conversion with a prominent CTA at the bottom of the page before the site footer.

## Content Zones
- Headline (e.g., "Ready for Better Water?")
- Subheadline (1-2 sentences of supporting text)
- Primary CTA button (e.g., "Get Your Free Quote")

## Visual Style
- High-contrast block with dark background or gradient
- Centered text alignment
- Rounded container with generous padding
- CTA button with hover animation
- Contrasting button color for visibility

## Dependencies
- `@/components/ui/button`
- `@/components/ui/container`

## Component API
```tsx
<CtaFooter />
```

## Accessibility
- Semantic section element with aria-labelledby
- High contrast ratio for text on colored background
- Focus-visible states for button
