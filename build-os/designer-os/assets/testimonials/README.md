## Locality Constraint (V2)

Testimonials must reference cities in **Seminole & Volusia counties, Florida** only:
- Sanford
- Altamonte Springs
- Lake Mary
- Deltona
- DeBary
- Daytona Beach
- Ormond Beach

**No out-of-state locations.**

## Requirements Summary

- **Type**: Real customer photos or professional stock headshots
- **Style**: Natural, approachable, trustworthy
- **Format**: JPG or WebP, optimized for web
- **Resolution**: Minimum 200×200px, 512×512 preferred
- **Aspect ratio**: 1:1 (square)
- **Background**: Neutral or lightly blurred

## File Naming (Implementation)

- `testimonial-1.jpg` — 512×512
- `testimonial-2.jpg` — 512×512
- `testimonial-3.jpg` — 512×512

## QC Notes

- Similar lighting and crop across all headshots
- Calm, approachable expressions
- No visible AI distortions

Do not place actual image files in this folder.
All real assets must live in `public/assets/`.
